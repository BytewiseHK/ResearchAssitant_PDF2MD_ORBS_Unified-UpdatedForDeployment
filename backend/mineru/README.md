# Research Assistant backend (cloud PDF)

This directory contains the **FastAPI** app (`main.py`) and **`mineru_net_client.py`**, which calls [MinerU.net](https://mineru.net) to convert PDFs to Markdown.

There is **no** vendored MinerU / PyTorch stack in this branch (`lightweight-cloud`). Set **`MINERU_API_KEY`** in the environment.
